# perl-maxcdn
MaxCDN API for perl

# MaxCDN REST Web Services Perl Client

## Requirements (open cpan)

`install JSON`

`install Net::OAuth`

`install LWP::UserAgent`

`install URI`

`install Data::Dumper`

`install IO::Socket::SSL`

`install LWP::Protocol::https`


## Installation
`wget https://github.com/idabic/perl-maxcdn/archive/master.zip`

`unzip master.zip`

`cd perl-maxcdn-master/`
